# remind.kager.io
A personal reminder webapp that plugs into an external messaging app using sockets to provide a real-time web experience.

# Architecture
![preview image](http://www.kager.io/uploads/minor/real-time-web/schematic.jpg)

## Dependencies
- Docker
- ArangoDB
- NodeJS
- NPM (package.json)
    - Debug
    - Express
    - Handlebars-express
    - ArangoJS
    - Socket.io
